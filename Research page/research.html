Navigation bar

For the navigation bar, we really thought of a clean navigation bar with simple color shades. After observing different websites, the JoeCoffee website got to be the most similar one among the others. So, we utilized some tags for navigation bar:
<nav> for a section for containing navigating elements
<ul> for creating an unordered list
<li> for listing elements
<img> for image of logo.
